OLD STANDARD — Backend Engineering (2025)
۱) API Design

REST اصولی + نسخه‌گذاری درست (/v1/...)

OpenAPI/Swagger مستندات رسمی

استفاده از HTTP status code دقیق

Idempotency برای requestهای حساس

Error format استاندارد (Problem Details RFC 9457)

اگر API داری بدون اینا، تیم‌های قوی جدي نمی‌گیرنش.

۲) Backend Frameworks

FastAPI برای Python → سریع، استاندارد، تایپ‌محور، بهترین انتخاب 2025

Django REST + DRF → وقتی سیستم خیلی بزرگ و سازمانی باشه

در دنیا الان FastAPI = golden standard برای سرویس‌های مدرن

۳) Code Quality

Type annotations کامل (Python typing)

100% independent modules

Zero business logic inside views → همه چیز در service layer

اگر بیزنس‌لاجیک داخل view هست، همون‌جا معلومه تیم ضعیفه.
